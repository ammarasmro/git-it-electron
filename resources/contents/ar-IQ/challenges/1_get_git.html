Git

Git هو برنامج مفتوح المصدر مجاني (مصدر الشفرة ل مفتوح الرخصة. كتب من قبل Linus Torvalds الذي أيضا كتب شفرة ال Kernel لنظام التشغيل ال Linux

Git هو برنامج لتعقب التغييرات على مدة الزمن, هذا يعرف في عالم البرمجة ب إدارة الاصدارات اذا كنت قد تتبعت التغييرات في احد برامج تحرير النص اذا فهذا المفهوم مألوف بالنسبة لك!

تنصيب Git

نحن نوصي بتنصيب Git على حاسوبك عن طريق تحميل برنامج GitHub Desktop. لن نستخدم هذا البرنامج في Git-it لأننا نتعلم ال terminal, ولكن هذا البرنامج يحتوي على Git أصلا وهو اسهل طريقة لتنصيبها على جميع أنظمة التشغيل.

  • Windows: استخدم Git Shell ك terminal
  • Mac: افتح GitHub Desktop ومن قائمة ال Preferences, اضغط على خيار command line tools install. استخدم terminal ك ال terminal.

لديك Git أصلا او لست متاكدا؟ اكتب git --version في ال terminal. اذا ظهر رقم اصدار 1.7.10, او احدث, فانك جاهز. لمعلومات أخرى, اذهب الى Git website.

Git برنامج

برنامج GitHub Desktop يستطيع عمل الكثير من الأشياء مع Git ولكن ليس كل شيء, لهذا فان تعلم ال terminal مهم. لكن عندما تنتهي من ذلك, ستسعد لكونك تملك برنامج سطح المكتب لكونه يرتب معلومات مشاريعك بطريقة حسنة للنظر, مثل موقع GitHub.

Git وحدها ليست مثل باقي البرامج على حاسوبك. على الأرجح انك لن ترة ايقونة على سطح المكتب, ولكنها موجودة ويمكنك وصولها في أي وقت عن طريق ال terminal او احد برامج Git.

ضبط Git

بعد تنصيب GitHub Desktop, ومعها Git, افتح ال terminal. يمكن التحقق من كون Git منصبة بإصدار الايعاز:

git --version

هذا سيرجع اصدار Git على حاسوبك. النتيجة ستشبه:

git version 1.9.1

لاحقا, اضبط Git لكي تعرف ان تربط عملك معك

اضبط اسمك:

git config --global user.name "Your Name"

الآن اضبط عنوان بريدك الالكتروني:

git config --global user.email "youremail@example.com"

لقد انتهيت من اول مهمة لك! اضغط على زر "تحقق" لتتحقق من المهمة
