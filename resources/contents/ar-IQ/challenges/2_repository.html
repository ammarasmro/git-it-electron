مستودعات البيانات

>مستودع البيانات هي مجموعة من الأصناف المتعلقة ببعضها. في حالتنا, عند كتابة شفرة البرامج, هي مجموعة من الملفات المتعلقة بمشروع البرنامج. يمكنك تخيلها على انها مجلد للمشروع يحتوي على جميع الملفات ذات صلة في داخلها. في الحقيقة, هذا ما ستبدو عليه على حاسوبك.

انت تخبر Git ما هو مشروعك و Git ستبدأ بتعقب كل التغييرات في ذلك المجلد. هذا ما يجعلها مستودع بيانات Git: مجلد يحتوي على عناصر تراقب تغييراتها عن طريق Git. بالإضافة الى ذلك, من عمل التغيير ومتى امران اخران يتتبعان عن طريق Git. في مجال البرمجة, تتبع التغييرات بهذه الطريقة يسمى إدارة الاصدارات

استخدام ال terminal

ال Terminal, او Bash, هو طريقة لايعاز الحاسوب لعمل مهام عن طريق كتابة الأوامر بدلا من الضغط على ايقونات البرامج. يمكنك تغيير أسماء الملفات, فتح ملفات, انشاء مجلدات جديدة, التحرك بين المجلدات, واكثر عن طريق تشغيل أوامر مكتوبة. يمكنك أيضا استخدام محرر نصوص لكتابة الشفرات, مثل Vim, في داخل ال terminal بدون مغادرته.

بالإضافة الى التحرك داخل الحاسوب, يمكنك أيضا استخدام برامج من داخل ال Terminal, التي تملك ما يسمى command-line interface (CLI), ما يعني انها تعمل باستخدام أوامر من ال Terminal. Git هي واحدة من هذه البرامج. الجزء الأول من الايعاز يوعز الحاسوب انك تتحدث الى Git. الأقسام التي تليه هي الأوامر والاختيارات التي تريد Git ان تتعامل معها.

في Git-it سوف تتعلم بعض الأوامر البسيطة, بالإضافة الى أوامر Git, للتحرك داخل الحاسوب.

انشاء مستودع بيانات

سوف تنشئ مجلد مشروع جديد ثم سوف تنشئ متسودع بيانات Git فيه. سوف نطلع على المجلد “hello-world” في هذه المهمة, لكن يمكنك استخدام أي اسم اخر في حال كان لديك مجلد بهذا الاسم أصلا. هذا الاسم عادة يطلع على اول مشروع.

حان وقت فتح ال Terminal

في نافذة ال Terminal, اكتب هذه الأوامر, واحدا تلو الاخر, ثم اضغط على enter/return بعد كل امر

أولا, افتح مجلدا جديدا: :
نصيحة: mkdir تعني make directory

mkdir hello-world

ثم, تحرك الى داخل المجلد: :
نصيحة: cd تعني change directory

cd hello-world

في النهاية, اخبر Git لكي تبتدأ, تبدأ عملية التعقب, للمجلد الحالي:

git init

الامر الأخير يجب ان يرجع جملة تبدأ ب "Initialized empty Git repository". الأوامر الأخرى لن ترجع أي شيء.

لقد فعلتها! اذا كنت تريد ان تتاكد انه مستودع بيانات Git, اكتب git status, واذا لم ترجع 'fatal: Not a git repository...', فانك قد نجحت!
