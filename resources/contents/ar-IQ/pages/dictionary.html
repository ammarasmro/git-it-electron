في ما يلي اوامر ال terminal وال Git التي يتم استخدامها في مهام Git-it

Bash/Terminal اساسيات

  • انشأ دليلا جديدا
    • mkdir <اسم الدليل>
  • انتقل الى دليل عمل موجود (بمعنى اخر غير دليل العمل الحالي)
    • cd <اسم الدليل>
  • اسرد محتويات الدليل
    • ls

ضبط Git

  • التحقق من اصدار Git
    • git --version
  • ضبط اسمك
    • git config --global user.name "اسمك"
  • ضبط عنوان بريدك الالكتروني
    • git config --global user.email youremail@example.com
  • ضبط حسابك الخاص في GitHub
    • git config --global user.username uSeRnAmE

Git اساسيات

  • تشغيل Git في دليل عمل
    • git init
  • التحقق من حالة التغييرات في مستودعة البيانات
    • git status
  • اظهار التغييرات في الملفات
    • git diff
  • اظافة التغييرات في الملف الى مرحلة الترحيل
    • git add <اسم الملف>
  • لاظافة جميع الملفات الى مرحلة الترحيل
    • git add .
  • لتثبيت التعديلات على الملفات مع رسالة مختصرة تشرح التغيير
    • git commit -m "your commit message"
  • انسخ مستودعة بيانات الى حاسوبك الخاص
    • git clone <الرابط>

افرع Git

  • انشاء فرع جديد
    • git branch <اسم الفرع>
  • للتحول الى فرع اخر
    • git checkout <اسم الفرع>
  • يمكنك انشاء فرع والتحول اليه في ايعاز مختصر في سطر واحد
    • git checkout -b <اسم الفرع>
  • اعرض جميع الفروع
    • git branch
  • اعد تسمية الفرع الحالي
    • git branch -m <اسم الفرع الجديد>

Git بعيد

  • اضافة روابط بعيدة
    • git remote add <اسم الرابط البعيد> <الرابط>
  • اضبط رابط لبعيد
    • git remote set-url <اسم الرابط البعيد> <الرابط>
  • اظهر الروابط البعيدة
    • git remote -v

سحب التغييرات في Git

  • سحب التغييرات
    • git pull
  • سحب التغييرات من فرع بعيد
    • git pull <اسم الرابط البعيد> <اسم الفرع البعيد>
  • انظر الى التغييرات في الفرع البعيد قبل سحبها
    • git fetch --dry-run

الادخال & الدمج في Git

  • ادخال التغييرات
    • git push <اسم الرابط البعيد> <اسم الفرع>
  • دمج فرع مع الفرع الحالي
    • git merge <اسم الرابط>

حذف الفروع المحلية والبعيدة في Git

  • لحذف فرع محلي
    • git branch -D <اسم الفرع>
  • لحذف رابط بعيد
    • git push <اسم الرابط البعيد> --delete <اسم الفرع>
