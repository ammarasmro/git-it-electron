GitHub Desktop

من المرجح انك قمت بتنصيب هذا البرنامج قبل البداية ب Git-it لكن ان لم تفعل ذلك, هذا رابط التحميل.

برنامج GitHub Desktop يسمح لك عمل اغلب ما يمكنك عمله في ال terminal وفي بعض الاحيان هو اسهل للقراءة. شخصيا, انا استخدمه للنظر الى اي diff كبير. تغييراتك اسهل للرؤية بهذه الطريقة.

ماذا بعد؟

لا بد انك تملك شعورا جيدا لكونك سيطرت على الاساسيات, هذه بعض الافكار لمشاريع لكي تستمر!

  • اعمل تشعيبا لهذه لعمل موقع شخصي (يتطلب HTML & CSS)
  • معرض المواقع القابلة للتشعيب (يتطلب HTML & CSS)
  • انشأ مستودع بيانات وافتح مشاكل في to-do lists.
  • تصفح مشاريع GitHub لتجد ما يثير اهتمامك.
