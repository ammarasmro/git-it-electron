حول Git-it

Git-it يعلم بدائيات Git التي يستخدمها المبرمجين يوميا مع مجرى عمل ال GitHub, بالطريقة التي يتعاون بها المبرمجين على GitHub.

بعض التحديات تتطلب اتصال بالانترنت, فيجب عليك ان تكون متصل بالانترنت لكي تكمل Git-it. لكن يمكنك دائما فتح البرنامج والاتطلاع على التحديات والمعلومات بدون انترنت.

محيط تطوير حقيقي

Git-it يستخدم ال terminal الحقيقي, Git و GitHub (بدلا عن عمل محاكاة). عندما تنتهي, سيكون لديك مستودعات بيانات حقيقية على حساب ال GitHub الخاص بك ومربعات خضراء في مخطط مساهماتك!

الشفرة التي عملت Git-it

انشأ هذا البرنامج باستخدام HTML, CSS و JS ويستخدم مكتبةElectron. كل البرنامج ل Git-it موجودة في مستودعة البيانات على GitHub.
